# @rnx-kit/types-transformer-esbuild

[![Build](https://github.com/microsoft/rnx-kit/actions/workflows/build.yml/badge.svg)](https://github.com/microsoft/rnx-kit/actions/workflows/build.yml)
[![npm version](https://img.shields.io/npm/v/@rnx-kit/types-transformer-esbuild)](https://www.npmjs.com/package/@rnx-kit/types-transformer-esbuild)

Type definitions for
[@rnx-kit/metro-transformer-esbuild](https://github.com/microsoft/rnx-kit/tree/main/packages/metro-transformer-esbuild).

Published separately so consumers can reference the types without depending on
the full implementation package.

## Installation

```sh
yarn add @rnx-kit/types-transformer-esbuild --dev
```

or if you're using npm

```sh
npm add --save-dev @rnx-kit/types-transformer-esbuild
```
