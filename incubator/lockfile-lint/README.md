`@rnx-kit/lockfile-lint` is a standalone tool for scanning your lockfiles for
potential issues.

Supported lockfile formats:

- [Yarn v1](https://classic.yarnpkg.com/en/docs/yarn-lock)
- [Yarn v2+](https://yarnpkg.com/advanced/changelog#200)

## Installation

```sh
yarn add @rnx-kit/lockfile-lint --dev
```

## Usage

```sh
yarn lint-lockfile
```
